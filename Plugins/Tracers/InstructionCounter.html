InstructionCounter¶

The InstructionCounter plugin counts the number of executed instructions along each execution path. Only instructions inside the modules of interest are counted.

Options¶

This plugin does not have any option.

Required Plugins¶

  • ExecutionTracer

Configuration Sample¶

pluginsConfig.InstructionCounter = {}
